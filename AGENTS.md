# ポモドーロタイマー

Pomodoro timer without sound.   Timeout is notified by visual feedback.

耳栓をしていてもわかるようにwindowをフラッシュさせて気がつかせるポモドーロタイマーです

1) ボタンを押すと25分のタイマーが始まります。
2) 25分の終わりにwindowの色をフラッシュさせます
3) ボタンを押すとフラッシュが止まり、5分のタイマーが開始します
4) 5分の終わりにwindowの色をフラッシュさせます
5) ボタンを押すとフラッシュが止まり、25分のタイマーが開始します。 (->2に戻る)

- ボタンはwindowいっぱいの大きさ
- フラッシュの色は選択できる。 白/黒 など
- フラッシュする領域はボタンの背景
- 25分・5分の計測中は秒単位で残り時間がボタンに表示される。

## 詳細要件

### フラッシュ動作

- フラッシュの間隔：2秒で明るく、2秒で暗くする（ゆっくり明るく・暗くする）

### 設定機能

- 設定は起動中も変更可能
- 歯車設定アイコンと設定画面を提供
- 設定ファイル: `~/.config/pomodoro/settings.ini` に保存

### 時間表示

- 表示フォーマット：`25:00`（分:秒）

### アプリケーション動作

- アプリ開始時は25分開始待ち。ボタンを押すと開始する
- アプリの終了はウィンドウを閉じる
- タイマー動作中はボタンを押すと一時停止
